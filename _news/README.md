# News Post Template

Create news items in `_news/` with filename format:

- `YYYY-MM-DD-short-slug.md`

Use this front matter:

```md
---
title: Short news title
category: Announcement # Announcement | Award | Event
date: 2026-03-01
---

One or two short paragraphs describing the update.
```

Guidelines:
- Keep title under 80 characters.
- Use `Announcement`, `Award`, or `Event` for `category`.
- Put key result in the first sentence.
